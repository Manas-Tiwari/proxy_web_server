# Proxy Web Server

A simple web proxy server that forwards client HTTP requests to target servers and returns the responses.

## Requirements
- Linux / macOS
- Bash
- GCC (or compatible C compiler)

## Build and Run
```bash
git clone <repo-url>
cd proxy_web_server
chmod +x build_and_run.sh
./build_and_run.sh
```
